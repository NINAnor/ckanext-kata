{{ _('The research data itself is not stored in Etsin. Rather, Etsin allows you to search for and browse datasets based on their metadata.') }}

{{ _('What is metadata?') }}

{{ _('Metadata is data about data. This is a record that provides information describing the data and how to access it. Firstly, metadata describes important features of the data such as authors, subjects, the fields of research the data pertains to, storage formats, and licenses.') }}

{{ _('However, the content of metadata varies depending on what kind of data is being described. The Etsin provides a common ground by employing a common set of metadata fields tailored for research data. In the interest of consistency with other services, the metadata conforms to the minimum metadata model defined by the Finnish National Research Data Initiative (TTA).') }}
